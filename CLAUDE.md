# Skill System

This repository contains skills for Claude Code. Skills are loaded from the `skills/` directory.

## Available Skills

- **yukti-demo-builder** — Build customized, interactive ScaleYukti WhatsApp engagement demos for any business from just a website URL

## How Skills Work

Each skill has its own directory under `skills/<skill-name>/` with a `SKILL.md` file defining the workflow. Reference files used by the skill are stored alongside it in `references/`.

Before starting any task, check if any registered skills are relevant.
